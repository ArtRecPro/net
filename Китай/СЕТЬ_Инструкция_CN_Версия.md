## 2. Код переключателя языков

### 2.1 HTML — добавить в навигацию (обе версии)

**Для русской версии (index.html):**

```html
<!-- Найти <nav> и добавить перед закрывающим </nav> -->
<div class="lang-switcher">
    <a href="index.html" class="lang-btn active">RU</a>
    <span class="lang-divider">/</span>
    <a href="cn/index.html" class="lang-btn">中文</a>
</div>
```

**Для китайской версии (cn/index.html):**

```html
<!-- Найти <nav> и добавить перед закрывающим </nav> -->
<div class="lang-switcher">
    <a href="../index.html" class="lang-btn">RU</a>
    <span class="lang-divider">/</span>
    <a href="index.html" class="lang-btn active">中文</a>
</div>
```

## 3. Создание китайской версии

### 3.1 Скопировать index.html в cn/index.html

```bash
mkdir -p cn
cp index.html cn/index.html
```

### 3.2 Исправить пути в cn/index.html

Все относительные пути нужно обновить (добавить `../`):

```html
<!-- ДО -->
<link rel="stylesheet" href="css/style.css">
<img src="images/logo.png">

<!-- ПОСЛЕ -->
<link rel="stylesheet" href="../css/style.css">
<img src="../images/logo.png">
```

**Поиск и замена:**
- `href="css/` → `href="../css/`
- `src="images/` → `src="../images/`
- `href="index.html` → `href="../index.html` (для внутренних ссылок на русскую версию)

### 3.3 Заменить текст

Использовать файл **СЕТЬ_Перевод_CN.md** для замены всего текстового контента.

## 5. Важные замечания

### 5.1 Шрифты для китайского

Убедиться, что в CSS подключены китайские шрифты:

```css
body {
    font-family: 'Inter', 'Noto Sans SC', 'PingFang SC', 'Microsoft YaHei', sans-serif;
}
```

Или добавить Google Fonts:

```html
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@400;500;700&display=swap" rel="stylesheet">
```

### 5.2 Meta-теги для китайской версии

```html
<html lang="zh-CN">
<meta charset="UTF-8">
<title>网 — 微短剧</title>
<meta name="description" content="2000年代初的莫斯科。微短剧，62集。">
```

### 5.3 Якорные ссылки

Якоря в навигации (`#hero`, `#story` и т.д.) менять НЕ нужно — они работают одинаково.
