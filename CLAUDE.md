# Проект "Сеть"

**Питч-сайт микродрамы** о мобильном бизнесе в России 2000-х.

## Быстрый старт

1. Прочитай полный контекст: `docs/PROJECT.md`
2. Запусти сервер: `bash serve.sh`
3. Документация: `docs/SLIDES.md`, `docs/CONTRAJUR.md`, `docs/ADAPTIVE.md`

## Структура

- `/index.html` — русская версия
- `/cn/index.html` — китайская версия (переключатель языков RU/中文)
- `/styles.css`, `/script.js` — общие для обеих версий

## Правила

- **Отвечай на русском языке**
- **НЕ сокращай текст** из файлов пользователя
- **Адаптивность обязательна** — см. `docs/ADAPTIVE.md`
- После `git push` напомни: **Ctrl+Shift+R**
