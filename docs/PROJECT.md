# Проект "Сеть" — Контекст

## Описание

**"Сеть"** — питч-сайт микродрамы о становлении мобильного бизнеса в России 2000-х годов.

- **Жанр:** Микродрама (вертикальный формат 9:16)
- **Формат:** 62 серии × 1.5-2 минуты
- **Время действия:** 2002-2004, Москва
- **Главный герой:** Макс Шалаев — предприниматель, строящий сеть салонов связи

### Китайская версия
- **Путь:** `/cn/index.html`
- **URL:** `https://artrecpro.github.io/net/cn/`
- **Особенности:**

  - Использует общие `../styles.css` и `../script.js`

## Разделы сайта

1. **Hero** — Название, слоган, формат
2. **История** — Логлайн, синопсис, PDF-материалы
3. **Герои** — 4 персонажа (Макс, Катя, Олег Михайлович, Антон)
4. **Визуальный язык** — Референсы, цветовая драматургия, формат 9:16
5. **Ритм и эмоция** — Музыкальная концепция 2002-2004
6. **Режиссёрское видение** — Манифест, дримкаст, камера, локации, костюмы, эпоха
7. **Производство** — Хронометраж, структура, преимущества формата, аудитория
8. **Контакты** — Артём Оськин

## Технические особенности

### Версионирование
```html
<link rel="stylesheet" href="styles.css?v=178">
<script src="script.js?v=17"></script>
```
Увеличивать версию при изменениях для сброса кеша.

### Адаптивность (критично!)
- `background-attachment: fixed` НЕ работает на iOS — отключать для ≤1024px
- Breakpoints: 1024px, 768px, 480px, 360px
- Гамбургер-меню на ≤768px

### Слайдеры локаций
Конфигурация в `script.js`:
```javascript
const locationSliders = {
    gorbushka: { title: '...', images: [...] },
    magazin: { ... },
    // ...
};
```
